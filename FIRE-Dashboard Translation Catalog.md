## 8. Dynamic Text Templates

These strings are assembled programmatically and contain variable placeholders:

### Phase Descriptions

**Template Pattern:**
```
Phase 1 (FIRE → 59.5): X yrs, taxable only
Phase 2 (59.5 → SS): Y yrs, 401K unlocks  
Phase 3 (SS → age Z): Unrestricted withdrawal
```

**Translatable components:**
- "Phase 1", "Phase 2", "Phase 3"
- "FIRE → 59.5", "59.5 → SS"
- "taxable only"
- "401K unlocks"
- "Unrestricted withdrawal"

### Country Scenario Cards

Each country card displays:
- Flag emoji + country name
- "FIRE age: X" (calculated)
- FIRE number "$X"
- Badges: "ZH" (Chinese), "EN" (English), "TAX" (no income tax), "💰 Under $50K"

**Translatable badges:**
- "ZH" → keep as is (language code)
- "EN" → keep as is (language code)
- "TAX" → translate to meaning "No Foreign Income Tax"

### Snapshot Deltas

Row shows: "▲ +$XXX" (green) or "▼ -$XXX" (red)

**Translatable component:** The delta direction indicator might have localized representations

### Mortgage Impact Summary

**Dynamic HTML example:**
```html
<strong>✅ Buy verdict:</strong><br>
Renting: FIRE in 15 yrs · Buying: FIRE in 18 yrs<br>
Down + closing ($120K) invested at ~7% for 18yr → $485K (+$365K lost growth)
```

**Translatable components:**
- "Buy verdict" / "Rent verdict"
- "Renting: FIRE in X yrs"
- "Buying: FIRE in Y yrs"
- "Down + closing"
- "invested at ~Y% for Z yr"
- "lost growth"

### Coast FIRE Note

**Dynamic variations:**

**Translatable components:**

## Translation Notes

1. **Emoji:** Keep all emoji as-is (🔥, 💰, 📈, etc.). They are universal and do not translate.

2. **Abbreviations:**
   - "OBJ" = objective (could translate or keep as label)
   - "EST" = estimated (could translate or keep as label)
   - "Cr" = credits (in Social Security context)
   - "SS" = Social Security (commonly understood acronym)

3. **Financial Terms:**
   - "FIRE" = Financial Independence, Retire Early (keep acronym)
   - "Safe Withdrawal Rate (SWR)" = 4% rule concept
   - "Geo-arbitrage" = living in low-cost countries
   - "Coast FIRE" = stop saving, let compound growth finish job

4. **Arrow Notation:**
   - "FIRE → 59.5" = may need context for clarity in translation
   - "59.5 → SS" = 59.5 is age (US tax rules for early 401K access)
   - Keep → arrows for visual clarity

5. **Dynamic Values:**
   - $ amounts, %, years — these are numbers and don't translate
   - Date formats — currently "Apr 2026", "MMM DD, YYYY" — may need localization

6. **HTML Structure:**
   - All labels are in `<label>` tags (easy to locate in HTML)
   - Tooltips use `data-tip="..."` attribute
   - Card titles use `<div class="card-title">`
   - Section headers have emoji prefix (keep emoji)
